<!DOCTYPE html>
<html>
<head>
    <title>NBA Statistics</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>NBA Statistics</h1>
    <!-- Container for the latest game results -->
    <div id="latest-results">
        <h2>Latest Results</h2>
        <!-- This element will be populated with the latest game results using JavaScript -->
        <ul id="results-list"></ul>
    </div>
    <script src="https://d3js.org/d3.v5.min.js"></script>
    <script>
        // Fetch the latest game results from the Flask app using an AJAX request
        d3.json("/latest-results", function(error, data) {
            if (error) throw error;
            // Select the results list element
            var resultsList = d3.select("#results-list");
            // Append a list item for each game result
            data.forEach(function(d) {
                resultsList.append("li")
                    .text(d.homeTeam + " vs " + d.awayTeam + ": " + d.homeScore + "-" + d.awayScore);
            });
        });
    </script>
</body>
</html>

In this example, the homepage template includes an empty <ul> element with the ID results-list. This element is populated with the latest game results using an AJAX request to the Flask app and the D3.js library.
